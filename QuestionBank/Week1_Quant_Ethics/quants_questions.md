An investor puts down $40,000 in cash to purchase a property worth $200,000, borrowing the remaining $160,000 at an annual interest rate of 5%. After one year, the property value increases by 10%. Ignoring transaction costs and taxes, the investor's leveraged return on their cash investment is *closest* to:

- A) 10.0%
- B) 30.0%
- C) 50.0%


<details>
<summary>âœ… Answer & Explanation</summary>

  **Correct Answer: B**

  **Explanation:**

  Initial equity investment ($V_0$) = $40,000.
  Borrowed amount ($V_B$) = $160,000.
  Total investment value = $200,000.
  Return on total investment (r) = 10%.
  Increase in property value = $200,000 * 10% = $20,000.
  Borrowing cost ($r_B V_B$) = $160,000 * 5% = $8,000.
  Investor's profit = Increase in property value - Borrowing cost = $20,000 - $8,000 = $12,000.
  Leveraged return = Investor's profit / Initial equity investment
  Leveraged return = $12,000 / $40,000 = 0.30 or 30.0%.

  Alternatively using the formula \(R_L = \frac{r(V_0 + V_B) - r_B V_B}{V_0}\):
  \(R_L = \frac{0.10(40000 + 160000) - 0.05(160000)}{40000}\)
  \(R_L = \frac{0.10(200000) - 8000}{40000} = \frac{20000 - 8000}{40000} = \frac{12000}{40000} = 0.30\) or 30.0% .
